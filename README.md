# 💰 ExpenseWise – Personal Finance Intelligence

> Transforming money tracking into a **story-driven, AI-powered experience**

---

## 🚀 About the Project

ExpenseWise is an intelligent personal finance system that helps users:

- Track expenses
- Analyze spending behavior
- Get smart AI recommendations
- Monitor financial health

This project is built using:
- Flask (Backend)
- HTML, CSS, JS (Frontend)
- Machine Learning (Spending Predictions)

---

## 🔁 Project Evolution

This project originally started as:

### 👗 WardrobeIQ (Old Version)

A smart wardrobe management system focused on:
- Clothing tracking
- Cost-per-wear analysis
- Outfit insights

🧠 Key Learning:
- Data tracking systems
- UI structuring
- ML integration basics

---

### 💡 Why Transition?

WardrobeIQ evolved into ExpenseWise because:

- Finance tracking has **higher real-world impact**
- More scope for **AI + analytics**
- Better product-level scalability
